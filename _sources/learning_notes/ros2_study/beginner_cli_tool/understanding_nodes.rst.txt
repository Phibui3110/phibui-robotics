Understand Nodes
================
Goal: Learn about the function of nodes in ROS2, and the tools to interact with them.

Background
----------

1. The ROS2 graph

2. Nodes in ROS2

Tasks
-----
1. ros2 run
   
   .. code-block:: bash

        ros2 run <package_name> <executable_name>

e.g.
    
    .. code-block:: bash

        ros2 run turtlesim turtlesim_code

2. ros2 node list

    .. code-block:: bash
    
          ros2 node list

    Show you the names of all running nodes.
