Projects¶

This section documents robotics projects developed in Phibui Robotics Lab.

Projects focus on autonomous vehicle development using ROS2, embedded systems, and computer vision.

Main hardware platform:

  • Raspberry Pi 5

  • STM32 Nucleo F411RE

  • BNO055 IMU

  • Raspberry Pi Camera Module 3 Wide

Project categories include:

  • Autonomous vehicle development

  • Embedded firmware

  • Sensor integration

  • Simulation and testing
